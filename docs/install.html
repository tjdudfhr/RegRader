🏛️ RegRader 앱 설치

기기 확인 중...
📲

5초만에 앱 설치하기!

📱 Android (Chrome)

1️⃣ 아래 "앱 열기" 버튼 클릭

2️⃣ 주소창 옆 ⋮ 메뉴 클릭

3️⃣ "앱 설치" 또는 "홈 화면에 추가" 선택

4️⃣ "설치" 버튼 클릭

✅ 완료! 홈 화면에서 앱 실행

🍎 iPhone/iPad (Safari)

1️⃣ 아래 "앱 열기" 버튼 클릭

2️⃣ 하단 공유 버튼 ⬆️ 클릭

3️⃣ "홈 화면에 추가" 선택

4️⃣ "추가" 버튼 클릭

✅ 완료! 홈 화면에서 앱 실행

💻 데스크톱 (Chrome/Edge)

1️⃣ 아래 "앱 열기" 버튼 클릭

2️⃣ 주소창 끝 설치 아이콘 🔧 클릭

3️⃣ "설치" 버튼 클릭

✅ 완료! 앱이 자동 실행됩니다
