✨ 이번엔 진짜 간단해요!
아래 버튼 클릭하면 바로 이메일 프로그램이 열려요!
📮 클릭하면 이메일 프로그램 열림!
📌 사용법:
1. 위 버튼 클릭
2. 이메일 프로그램(Outlook, Gmail 등)이 자동으로 열림
3. 받는 사람 주소를 실제 이메일로 변경
4. 보내기 버튼 클릭!

🔧 아니면 이 방법도 있어요!

방법 1: Gmail에서 직접 보내기
1. Gmail 열기 (클릭!)
2. Gmail 로그인
3. 받는 사람 주소 수정
4. 보내기!
방법 2: 복사해서 이메일에 붙여넣기
아래 내용을 복사해서 이메일에 붙여넣으세요:
제목: 법령 모니터링 일일 리포트 안녕하세요, 법령 모니터링 시스템 일일 리포트입니다. [전체 현황] 총 매칭 법령 수: 259건 [분기별 현황] • 1분기: 119건 • 2분기: 61건 • 3분기: 64건 • 4분기: 15건 [직무별 현황] • 안전보건: 50건 • 환경: 45건 • 소방: 30건 • 산업안전: 20건 • 기타: 114건 [최근 개정 법령] 1. 산업안전보건법 시행령 2. 환경정책기본법 3. 소방시설법 시행규칙 이메일 테스트가 정상적으로 완료되었습니다. 감사합니다.
방법 3: 네이버 메일로 보내기
1. 네이버 메일 열기
2. 로그인 → 메일쓰기
3. 위 내용 복사해서 붙여넣기
4. 보내기!
